## 🌟 Project Overview

Eth Yahana Hotel website is built to cater to modern travelers, combining elegant aesthetics with functional features. It serves as a digital gateway for guests to explore room types, view amenities, and initiate bookings.

---

## ✨ Key Features

- **📅 Smart Booking System**: Integrated form for check-in and check-out date selection.
- **🏨 Elegant Room Showcases**: Detailed listings for Standard, Deluxe, Honeymoon, and Executive suites.
- **💎 Amenity Highlights**: Interactive sections showcasing hotel services and facilities.
- **💬 Social Proof**: Customer testimonials and reviews section for building trust.
- **📍 Location Integration**: Embedded Google Maps and easy-to-find contact details.
- **📱 Fully Responsive**: Optimized for high performance across all devices.

---

## 🛠️ Built With

- **CMS:** WordPress
- **Backend:** PHP
- **Database:** MySQL
- **Design:** Custom Themes & Responsive UI Frameworks

## 📂 Installation

1. Clone the repository to your local server (XAMPP/MAMP/WAMP) htdocs folder.
2. Create a new MySQL database.
3. Import the provided SQL dump (if available) or proceed with a fresh WordPress installation.
4. Configure the `wp-config.php` file with your database credentials.
5. Access the site via your local browser.
